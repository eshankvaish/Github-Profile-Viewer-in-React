# React Assignment
 - For setup, execute the following commands:
    `nvm use`
    `npm install`
    `npm start`

 - For blue mode:
    `REACT_APP_THEME=blue npm start`

 - For brown mode:
    `REACT_APP_THEME=brown npm start`

 - For executing test scripts:
    `npm run test`

 - For Storybook:
    `npm run storybook`
